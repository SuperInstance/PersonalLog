## Design Tips

### Visual Design

**Color Scheme:**
- Primary: Brand color (e.g., #007bff for blue)
- Secondary: Complementary color (e.g., #28a745 for green)
- Accent: Highlight color (e.g., #ffc107 for yellow)
- Background: White or light gray

**Typography:**
- Headlines: Bold, 24-32pt
- Subheadlines: Semibold, 18-20pt
- Body: Regular, 12-14pt
- Monospace: For code examples

**Layout:**
- Grid-based alignment
- Consistent spacing (8px, 16px, 24px)
- Visual hierarchy (size, weight, color)
- Whitespace for readability

### Content Design

**Headline:**
- Catchy but descriptive
- Include main benefit
- Under 10 words

**Key Features:**
- 5-7 features maximum
- Use checkmarks for visual appeal
- Keep descriptions concise

**Metrics:**
- Use real numbers
- Round to easy-to-read values
- Include comparisons (before/after)

**Call to Action:**
- Clear and direct
- Include next steps
- Add urgency if appropriate
