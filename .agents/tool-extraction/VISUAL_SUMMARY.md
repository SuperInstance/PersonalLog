## Decision Framework

### Extract Tool If:
- ✅ Independence score ≥ 7/10
- ✅ Clear target user base
- ✅ Can be tested independently
- ✅ Has < 20 PersonalLog dependencies
- ✅ Provides unique value (no direct competitors)

### Defer Tool If:
- ❌ Independence score < 5/10
- ❌ Tightly coupled to UI components
- ❌ Requires PersonalLog data schema
- ❌ Unclear value proposition

### Prioritize Tool If:
- ⭐⭐⭐ Independence ≥ 8/10 AND high demand
- ⭐⭐ Independence ≥ 7/10 AND unique capability
- ⭐ Independence ≥ 6/10 AND foundational value
