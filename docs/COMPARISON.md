## Detailed Comparisons

### PersonalLog vs ChatGPT

**ChatGPT Strengths:**
- Excellent AI model (GPT-4)
- Polished user interface
- Mobile apps
- Large community
- Regular updates

**PersonalLog Advantages:**
- **Multiple AI Providers**: Use OpenAI, Anthropic, Google, Mistral, and more
- **Local-First**: All data stored on your device, not in the cloud
- **Privacy**: No data collection or tracking
- **Knowledge Base**: Integrated semantic search and knowledge management
- **Data Export**: Export to 6 formats (JSON, Markdown, CSV, HTML, YAML, PDF)
- **Open Source**: MIT license, self-hostable
- **Cost**: Use your own API keys, potentially cheaper

**When to Choose ChatGPT:**
- You want the simplest possible experience
- You don't care about data privacy
- You don't need knowledge management
- You prefer cloud-based solutions

**When to Choose PersonalLog:**
- You want control over your data
- You need knowledge management
- You want to use multiple AI providers
- You value privacy
- You want to self-host
- You need advanced data export

## Migration Guide

### From ChatGPT/Claude.ai

**What You Can Export:**
- Conversations: Export to JSON/Markdown, then import
- API keys: Reuse your existing keys

**What You'll Gain:**
- Knowledge management
- Data portability
- Privacy
- Multiple providers

**Migration Steps:**
1. Export conversations from ChatGPT/Claude.ai
2. Import into PersonalLog using Data → Import
3. Configure your API keys in Settings → AI Providers
4. Start new conversations with AI contacts

### From Obsidian

**What You Can Export:**
- Notes: Export as Markdown, import to Knowledge
- Attachments: Re-import or link

**What You'll Gain:**
- Built-in AI conversations
- Semantic search
- Automated backups

**Migration Steps:**
1. Export Obsidian vault as Markdown
2. Import to PersonalLog Knowledge
3. Create AI contacts for different workflows
4. Use semantic search instead of link-based search

### From Notion/Roam Research

**What You Can Export:**
- Pages/Notes: Export as Markdown/CSV
- Databases: Export as CSV

**What You'll Gain:**
- Privacy (local-first)
- AI conversations
- Semantic search
- No subscription fees

**Migration Steps:**
1. Export data from Notion/Roam
2. Import to PersonalLog Knowledge
3. Organize with tags and collections
4. Set up AI contacts for assistance
