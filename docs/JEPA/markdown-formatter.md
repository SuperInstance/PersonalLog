```
│  • Clipboard copy (navigator.clipboard)                 │
└─────────────────────────────────────────────────────────┘
```

### Key Design Decisions

1. **No Toast Dependencies:** Used console.log instead of toast library to avoid adding new dependencies
2. **Multiple Button Layout:** Instead of dropdown menu, used separate buttons for clearer UX
3. **Status Management:** Comprehensive state tracking for each export operation
4. **Error Handling:** Try-catch blocks with user-friendly error messages
5. **Type Safety:** Full TypeScript strict mode compliance
6. **Accessibility:** ARIA labels on all interactive elements
7. **Flexibility:** Support for both transcript objects and message arrays

## Usage Examples

### Basic Usage with Messages

```typescript
import { ExportControls } from '@/components/jepa'

function TranscriptViewer({ messages, sessionId }) {
  return (
    <div>
      {/* Display transcript */}
      <div>{/* ... */}</div>

      {/* Export controls */}
      <ExportControls
        messages={messages}
        sessionId={sessionId}
        onExportComplete={(format) => console.log(`Exported as ${format}`)}
      />
    </div>
  )
}
```

### Advanced Usage with Options

```typescript
import { formatMessagesToMarkdown } from '@/lib/jepa'

const { markdown, metadata } = formatMessagesToMarkdown(
  messages,
  sessionId,
  {
    includeTimestamps: true,
    includeSpeakerNames: true,
    includeMetadata: true,
    timestampFormat: 'hh:mm:ss',
    separator: 'line',
    includeConfidence: false,
  }
)

console.log(`Words: ${metadata.totalWords}`)
console.log(`Reading time: ${metadata.estimatedReadingTime} minutes`)
console.log(`Speakers: ${metadata.speakers.join(', ')}`)
```

### Custom Speaker Names

```typescript
import { getSpeakerDisplayName } from '@/lib/jepa'

const customNames = new Map([
  ['user', 'John Doe'],
  ['assistant', 'Claude AI'],
  ['speaker-1', 'Guest Speaker'],
])

const name = getSpeakerDisplayName('speaker-1', customNames)
// Returns: "Guest Speaker"
```
