## Version Policy

This project follows [Semantic Versioning](https://semver.org/):

- **Major version (X.0.0)**: Breaking changes, major redesigns
- **Minor version (0.X.0)**: New features, backwards compatible
- **Patch version (0.0.X)**: Bug fixes, backwards compatible

### Example Version Changes:
- `1.0.0` → `2.0.0`: Breaking changes (users must update code)
- `1.0.0` → `1.1.0`: New features (drop-in upgrade)
- `1.0.0` → `1.0.1`: Bug fixes (drop-in upgrade)
